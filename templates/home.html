What’s NPC?

NPC is an advanced tool designed to classify with an explanation whether code is human-written or generated by ChatGPT. The main purpose of NPC is accurate classification across multiple test instances. It specifies the source of the code along with a detailed description. These descriptions highlight specific features and characteristics that determine whether the code is human-written or generated by ChatGPT.

Features

Detection of ChatGPT-Generated Code vs. Human-Written Code

• Uses a trained machine learning model to analyze source code and determine whether it was generated by an AI like ChatGPT or written by a human.
• The system scans the structure, syntax, and style of the code to identify telltale signs that differentiate between AI-generated code and human-created code, ensuring accurate classification.

Explanation System for Providing Insights

• Provides a detailed explanation of why the code was classified as either AI-generated or human-written.
• This explanation leverages in-context learning to highlight specific characteristics in the code and neighborhood samples, offering users a clear understanding of the decision process and enhancing the interpretability of the classification results.
